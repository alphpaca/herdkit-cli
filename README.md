A PHP monorepo management CLI tool.

## Requirements

- [Bun](https://bun.sh/) >= 1.0

## Installation

```bash
bun install
```

## Usage

```bash
# Development
bun run dev

# Build standalone binary
bun run build
./dist/herdkit
```

## Development

```bash
bun test              # Run tests
bun test --watch      # Watch mode
bun run typecheck     # Type check
```
